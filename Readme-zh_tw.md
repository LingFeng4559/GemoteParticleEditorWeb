# Gemote 粒子特效 3D 繪圖器

專為 Minecraft 插件 **GEmote** 打造的專業 3D 網頁繪圖工具。

## 📜 致謝與專案背景

本專案是基於 **[Ninepin123](https://github.com/Ninepin123)** 所開發的 [MythicMobs 粒子編輯器](https://github.com/Ninepin123/Ninepin123.github.io) 進行深度修改與重構而成。

**特別說明**：原始專案與本重構專案均採用 **AI 輔助生成**。若您在開發過程中遇到任何問題或 bug，歡迎在 [GitHub Issues](https://github.com/LingFeng4559/GemoteParticleEditorWeb/issues) 進行反饋。

原專案主要用於製作 MythicMobs 的技能特效。我們將其核心邏輯轉向支援 **GEmote** 的 YAML 格式，並加入了專屬設定與現代化的多語言介面。您也可以查看 `目錄/js/lang` 資料夾來添加或修改自定義的語言配置檔。

## 🚀 功能特色

- **3D 互動畫布**：支援 360 度旋轉、縮放與平移，從任何角度繪製特效。
- **多樣化繪圖工具**：
    - **筆刷 (Brush)**：在 3D 平面上隨手繪製。
    - **點 (Point)**：精確放置單個粒子。
    - **形狀 (Shapes)**：快速生成完美的矩形與圓形。
- **進階對稱系統 (NEW)**：
    - **獨立三軸鏡像**：X、Y、Z 軸可獨立開啟或疊加。
    - **自訂中心點 (Pivot)**：可自訂對稱中心，並支援「自動跟隨平台」功能。
    - **放射鏡像 (Radial Symmetry)**：支援繞三軸旋轉複製，具備均分與固定間隔模式。
- **粒子密度控制 (NEW)**：
    - **1:1 精確校正**：UI 數值、預覽效果與實際生成結果完全一致。
    - 即時預覽圖：在調整時可直觀看到粒子的分佈疏密感。
- **參考人物模型 (NEW)**：
    - 內建 1.8 單位高度的 Steve 模型，作為比例與空間感參考。
    - 支援 **100% 模式**、**幽靈模式 (Ghost)** 與 **隱藏模式**。
- **動畫系統**：支援逐粒子延遲（Tick Delay），內建即時動畫預覽。
- **Minecraft 1.21 兼容**：內建完整的粒子清單。
- **專案管理**：支援將作品儲存為 `.gemote3d` 格式以供後續編輯。

## 🛠️ 詳細使用說明

### 💻 運行環境準備
由於瀏覽器對 JavaScript 模組的安全限制，您**無法**直接雙擊開啟 `index.html`。
1.  **一鍵啟動 (Windows)**：直接雙擊資料夾中的 `run.bat`，系統會自動啟動 Python 伺服器並開啟瀏覽器。
2.  **手動啟動**：在目錄下執行 `python run.py` 或使用 VS Code 的 "Live Server"。

### 🖱️ 3D 畫布操作
-   **左鍵點擊/拖動**：執行當前工具（繪製、選取、放置鏡子等）。
-   **右鍵拖動**：旋轉攝影機視角。
-   **中鍵拖動 (或 Ctrl+左鍵)**：平移攝影機視角。
-   **滾輪滾動**：縮放畫布。
-   **中鍵點擊**：將攝影機目標重設回原點 (0,0,0)。
-   **Ctrl + Z**：全域撤銷（返回上一步）。

### 🎨 核心功能面板

#### 1. 繪圖工具 (Drawing Tools)
-   **相機模式**：純觀看模式，防止誤觸繪製。
-   **選擇模式**：點擊粒子群組可進行移動或刪除。支援「框選」多個群組。
-   **筆刷模式**：在當前 3D 平面上自由手繪。
-   **鏡子模式**：點擊兩次以定義一條「鏡像線」，之後繪製的所有粒子都會自動對稱生成。
-   **形狀工具**：快速建立完美的長方形或圓形（支援實心與空心）。

#### 2. 粒子設定 (Particle Settings)
-   **類型選擇**：內建 40+ 種 Minecraft 1.21 粒子。
-   **進階顏色控制** (僅限紅石粒子)：
    -   **全彩色盤**：直接選取色調。
    -   **亮度滑桿**：快速調整顏色深淺。
    -   **RGB 輸入**：手動輸入 0-255 的精確數值。
    -   *以上三者具備「雙向同步」機制，修改任一項皆會連動。*

#### 3. 動畫系統 (Animation)
-   **啟用動畫**：勾選後，新繪製的筆劃將帶有延遲，依序播放。
-   **預覽動畫**：點擊「預覽動畫」按鈕可直接在 3D 畫面中觀看動態效果。
-   **群組覆寫**：選中特定群組後，可為其設定獨立的播放速度 (Tick 延遲)。

#### 4. 專案管理 (Project Management)
-   **儲存專案**：將目前的設計匯出為 `.gemote3d` 格式（JSON），方便日後繼續編輯。
-   **特效 ID**：此 ID 將決定匯出的 YML 檔案名稱。

### 🔄 完整工作流程
1.  **命名**：在「專案管理」設定您的特效 ID（例如 `FireRing`）。
2.  **選色**：切換到「紅石」粒子並利用進階控制面板調配您喜歡的顏色。
3.  **佈局**：調整「繪圖高度」與「平面旋轉」來定位您的繪圖空間。
4.  **繪製**：使用筆刷或形狀工具建立圖案。
5.  **測試**：點擊「預覽動畫」確認動態是否符合預期。
6.  **匯出**：點擊「生成程式碼」，將下載的 `.yml` 放入伺服器 `plugins/GEmote/emotes/` 目錄。
7.  **套用**：在伺服器輸入 `/gemote reload`，接著輸入 `/gemote FireRing` 即可展示！
