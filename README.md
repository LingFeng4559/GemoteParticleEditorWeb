## 🚀 快速開始 (Quick Start)

### Windows
1. 下載專案並解壓縮。
2. 雙擊 `run.bat` 即可啟動並自動開啟瀏覽器。

### Other Platforms
1. 確保已安裝 Python。
2. 執行 `python run.py`。
3. 在瀏覽器開啟 `http://localhost:8000`。
