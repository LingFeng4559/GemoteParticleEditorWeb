# Gemote 粒子特效 3D 绘图器

專為 Minecraft 插件 **GEmote** 打造的專業 3D 网页绘图工具。

## 📜 致谢与项目背景

本项目是基于 **[Ninepin123](https://github.com/Ninepin123)** 所开发的 [MythicMobs 粒子编辑器](https://github.com/Ninepin123/Ninepin123.github.io) 进行深度修改与重构而成。

**特别说明**：原始项目与本重构项目均采用 **AI 辅助生成**。若您在开发过程中遇到任何问题或 bug，欢迎在 [GitHub Issues](https://github.com/WeiChia/GemoteWeb/issues) 进行反馈。

原项目主要用于制作 MythicMobs 的技能特效。我们将视角核心逻辑转向支持 **GEmote** 的 YAML 格式，并加入了专属设定与现代化的多语言界面。您也可以查看 `目录/js/lang` 文件夹来添加或修改自定义的语言配置文件。

## 🚀 功能特色

- **3D 互动画布**：支持 360 度旋转、缩放与平移，从任何角度绘制特效。
- **多样化绘图工具**：
    - **笔刷 (Brush)**：在 3D 平面上随手绘制。
    - **点 (Point)**：精确放置单个粒子。
    - **形状 (Shapes)**：快速生成完美的矩形与圆形。
- **进阶对称系统 (NEW)**：
    - **独立三轴镜像**：X、Y、Z 轴可独立开启或叠加。
    - **自定义中心点 (Pivot)**：可自定义对称中心，并支持“自动跟随平台”功能。
    - **放射镜像 (Radial Symmetry)**：支持绕三轴旋转复制，具备均分与固定间隔模式。
- **粒子密度控制 (NEW)**：
    - **1:1 精确校正**：UI 数值、预览效果与实际生成结果完全一致。
    - 即时预览图：在调整时可直观看到粒子的分布疏密感。
- **参考人物模型 (NEW)**：
    - 内置 1.8 单位高度的 Steve 模型，作为比例与空间感参考。
    - 支持 **100% 模式**、**幽灵模式 (Ghost)** 与 **隐藏模式**。
- **动画系统**：支持逐粒子延迟（Tick Delay），内置即时动画预览。
- **Minecraft 1.21 兼容**：内置完整的粒子清单。
- **项目管理**：支持将作品保存为 `.gemote3d` 格式以供后续编辑。

## 🛠️ 详细使用说明

### 💻 运行环境准备
由于浏览器对 JavaScript 模块的安全限制，您**无法**直接双击开启 `index.html`。
1.  **一键启动 (Windows)**：直接双击文件夹中的 `run.bat`，系统会自动启动 Python 服务器并开启浏览器。
2.  **手动启动**：在目录下执行 `python run.py` 或使用 VS Code 的 "Live Server"。

### 🖱️ 3D 画布操作
-   **左键点击/拖动**：执行当前工具（绘制、选取、放置镜子等）。
-   **右键拖动**：旋转摄影机视角。
-   **中键拖动 (或 Ctrl+左键)**：平移摄影机视角。
-   **滚轮滚动**：缩放画布。
-   **中键点击**：将摄影机目标重设回原点 (0,0,0)。
-   **Ctrl + Z**：全局撤销（返回上一步）。

### 🎨 核心功能面板

#### 1. 绘图工具 (Drawing Tools)
-   **相机模式**：纯观看模式，防止误触绘制。
-   **选择模式**：点击粒子群组可进行移动或删除。支持“框选”多个群组。
-   **笔刷模式**：在当前 3D 平面上自由手绘。
-   **镜子模式**：点击两次以定义一条“镜像线”，之后绘制的所有粒子都会自动对称生成。
-   **形状工具**：快速建立完美的长方形或圆形（支持实心与空心）。

#### 2. 粒子设定 (Particle Settings)
-   **类型选择**：内置 40+ 种 Minecraft 1.21 粒子。
-   **进阶颜色控制** (仅限红石粒子)：
    -   **全彩色盘**：直接选取色调。
    -   **亮度滑块**：快速调整颜色深浅。
    -   **RGB 输入**：手动输入 0-255 的精确数值。
    -   *以上三者具备“双向同步”机制，修改任一項皆会联动。*

#### 3. 动画系统 (Animation)
-   **启用动画**：勾选后，新绘制的笔划将带有延迟，依序播放。
-   **预览动画**：点击“预览动画”按钮可直接在 3D 画面中观看动态效果。
-   **群组覆写**：选中特定群组后，可为其设定独立的播放速度 (Tick 延迟)。

#### 4. 项目管理 (Project Management)
-   **保存项目**：将目前的设计导出为 `.gemote3d` 格式（JSON），方便日后继续编辑。
-   **特效 ID**：此 ID 将决定导出的 YML 文件名称。

### 🔄 完整工作流程
1.  **命名**：在“项目管理”设定您的特效 ID（例如 `FireRing`）。
2.  **选色**：切换到“红石”粒子并利用进阶控制面板调配您喜欢的颜色。
3.  **布局**：调整“绘图高度”与“平面旋转”来定位您的绘图空间。
4.  **绘制**：使用笔刷或形状工具建立图案。
5.  **测试**：点击“预览动画”确认动态是否符合预期。
6.  **导出**：点击“生成代码”，将下载的 `.yml` 放入服务器 `plugins/GEmote/emotes/` 目录。
7.  **套用**：在服务器输入 `/gemote reload`，接着输入 `/gemote FireRing` 即可展示！
